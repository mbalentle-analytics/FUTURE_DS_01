# Sales Analysis Dashboard - Future Interns Task

A client-ready sales analysis project built as part of the Future Interns data analytics internship task.
Goal: Use real business sales data to answer strategic questions and provide actionable recommendations.

### **1. Project Overview**
This project analyzes $2.29M in sales data to help a business understand performance and decide where to grow faster.
Instead of just creating charts, the focus was on delivering insights a business owner, startup founder, or analytics client could act on.

**Business Questions Answered:**
1. Which products generate the most revenue?
2. How do sales change over time?
3. Which categories or regions are most profitable?
4. Where should the business focus to grow faster?

### **2. Key Insights**
1. **Revenue & Margin**: $2.29M total sales → $286,397 revenue → 12% profit margin
2. **Top Products**: Phones drove the highest revenue at $330,007. Top 3 products: Phones, Chairs, Storage
3. **Category Performance**: Technology + Office Supplies = 68% of sales. Furniture = 32%
4. **Shipping Analysis**: Standard Class = 58% of orders. Shows clear cost vs delivery speed tradeoff
5. **Regional/City View**: Sales vary significantly by city. Springfield and Seattle are top performers

### **3. Actionable Recommendations**
Based on the analysis, I recommend:
1. **Focus marketing + inventory** on Technology category and top products like Phones
2. **Review pricing/costs** in Furniture category to improve the 12% margin
3. **Optimize shipping strategy** by analyzing if Standard Class delays impact customer retention
4. **Target high-performing cities** like Springfield/Seattle for localized campaigns

### **4. Tools Used**
- **Excel**: Data cleaning, pivot tables, KPI calculations, dashboard design
- **Data Analysis Skills**: Logical thinking, business questioning, insight → recommendation flow

No advanced coding required. Built to show that curiosity + structured thinking are the core of analytics.
